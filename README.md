# 🏋️‍♂️ Gym Management System

A simple console-based Gym Management System developed in **C++** using **Object Oriented Programming (OOP)** concepts. This application allows users to manage gym members, track attendance, and calculate total fees through a menu-driven interface.

---

## 📖 Project Overview

The purpose of this project is to demonstrate the practical implementation of OOP principles such as classes, objects, encapsulation, and constructors. The system simulates basic gym operations by allowing users to add members, mark attendance, and calculate fees based on age and attendance status.

This project is designed as a beginner-friendly application to understand how real-world systems can be modeled using C++.

---

## ✨ Features

- ➕ Add new gym members
- ✅ Mark attendance
- 💰 Calculate total fees
- 📋 Menu-driven console interface
- 🧱 Uses Object Oriented Programming principles
- 📦 Dynamic storage using STL vector

---

## 🛠️ Tech Stack

- **Language:** C++
- **Concepts Used:** Object Oriented Programming (OOP)
- **Libraries:** iostream, vector

---

## ⚙️ How It Works

1. The program displays a menu with options.
2. Users can add members by entering name and age.
3. Attendance can be marked by searching the member name.
4. Fees are calculated based on age:
   - Age > 20 → Fee = 200
   - Age ≤ 20 → Fee = 100
5. Total fees are displayed for members who attended.
